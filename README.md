## Description
BMI Calculator App is a basic Android application developed using **Java** in **Android Studio**.  
The app allows users to enter their height and weight to calculate the **Body Mass Index (BMI)** and displays the result in a simple and user-friendly interface.  
This project is created as part of my Android development learning journey.

## Features
- Input height and weight  
- Calculate BMI value  
- Display BMI result  
- Simple and clean UI  

## Tech Stack
- Java  
- Android Studio  
- XML (UI Design)  

## Learning Outcomes
- Handling user input in Android  
- Implementing basic calculations in Java  
- Understanding Android activity lifecycle  
- Designing UI using XML
